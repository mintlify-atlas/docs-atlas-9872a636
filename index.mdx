leaftext is a desktop Markdown reader for macOS, Windows, and Linux. Open a file with `Ctrl+O` / `Cmd+O` and read it — no distractions, no configuration required. It renders CommonMark and GitHub Flavored Markdown faithfully, with syntax-highlighted code blocks, Mermaid diagrams, math, alert callouts, footnotes, and emoji.

## What leaftext does

leaftext is purpose-built for reading Markdown — not editing it. It renders the full CommonMark and GFM specification plus GitHub extras, presents documents in a clean reading view, and stays out of your way.

## Get started

<Steps>
  <Step title="Install leaftext">
    Download the release for your platform from [GitHub Releases](https://github.com/ryanallen/leaftext/releases) and follow the [installation guide](/installation).
  </Step>
  <Step title="Open a Markdown file">
    Press `Ctrl+O` (Windows/Linux) or `Cmd+O` (macOS) to open a file picker, or drag and drop a `.md` file onto the window.
  </Step>
  <Step title="Explore the interface">
    Use the tab bar for multi-document reading, the minimap rail for quick navigation, and the library pane to browse your Markdown files.
  </Step>
  <Step title="Adjust settings">
    Open **Settings** in the app bar to choose a theme, toggle the minimap, and set your preferred language.
  </Step>
</Steps>
